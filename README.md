# Required Packages

    - Python 3.9
    - pytorch 1.12.0
        can be installed following the instruction at
        https://pytorch.org/get-started/locally/
    - matplotlib 
        python -m pip install -U matplotlib
    - imageio
        python -m pip install -U imageio
    - tqdm
        pip install tqdm
